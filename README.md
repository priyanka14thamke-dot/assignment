# Gutenberg Project

A React application for browsing and exploring books 

## Overview

This web application allows users to search, browse, and view detailed information about books. It provides a clean, modern interface for discovering classic literature and public domain books.

## Features

- **Home Page**: Landing page with navigation to different sections
- **Books List**: Browse and search through the collection of books
- **Book Details**: View detailed information about individual 

books including:
  - Title and author information
  - Book metadata (subjects, formats, languages)
  - Download options and links
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technologies Used

- **React 18** - Frontend framework
- **TypeScript** - Type-safe JavaScript
- **React Router** - Client-side routing
- **Axios** - HTTP client for API requests
- **CSS3** - Styling with modern CSS features

## Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager
